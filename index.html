EC Frontend-utvecklare

Med YH-utbildning kommer du snabbt i kontakt med arbetslivet och lär dig det senaste inom frontend. Kraven på kunskap inom JavaScript blir allt större, det räcker i dagsläget inte bara med att kunna CSS och jQuery för att lyckas som frontend utvecklare. God förståelse för design och användarvänlighet är också värdefullt, och programmering blir allt viktigare i takt med att webbläsare börjar användas som en seriös programmeringsplattform.
